== Description ==

3D Avatar Creator tool which help you to configure and make your avatar customizable in 3D on your user profile. 

If 3d avatar is not configured, the plugin uses the standard Gravatar image to create a 3D figure with Gravatar colors.

It is also possible to customize the avatar by uploading a photo and choose your gender.

The avatar has its own skeleton and is compatible with readyplayer avatars in order to add movements and iterations. 

The aim is to affiliate more users with this additional tool.

This plugin use 3rd party service of Avatar 3D Creator from the website: https://avatar3dcreator.com 

Avatar 3D Creator grants the use of this service completely free of charge and for the diffusion on the web.

There is also a version of this plugin with the ability to configure fully customizable 3D character for each user and their avatar image profile.

Including the possibility to download the created avatars. 

A new experience for your users instead of an anonymous Gravatar image. 

For more information: <a href="https://avatar3dcreator.com/wordpress-plugin-3d-avatar-user-profile/">https://avatar3dcreator.com/wordpress-plugin-3d-avatar-user-profile/</a>

== Installation ==

This section describes how to install the plugin and get it working.

In WordPress:

    Go to Plugins > Add New > search for :: 3D Avatar User Profile ::
    Press Install Now for the Show notice or message on admin area plugin
    Press Activate Plugin

To install manually instead:

    Download the directory with the plugin files and unzip it from:

	https://wordpress.org/plugins/3d-avatar-user-profile/
	
    Upload the Avatar-3d-creator directory to the /wp-content/plugins/ directory
    Activate the plugin through the Plugins menu in WordPress
